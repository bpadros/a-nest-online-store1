## Description

This is a e-commerce website where you can buy articles in the products page , also you can create and user and login.As an admin you can upload products to the website.

## This site was built with:

+ Front-end : HBS library
+ Back-end : Nest.js framework

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
